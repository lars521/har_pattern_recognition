## 1. 标定目的与原理

MEMS 惯性/磁传感器在出厂时存在系统误差：

**加速度计误差**：

- **零偏（Bias）**：即使无加速度输入，传感器也会输出非零值
- **刻度因子误差**：传感器输出与实际物理量不成正比
- **轴间非正交**：三个测量轴并非完全垂直

**磁力计误差**：

- **硬铁干扰**：PCB 上的铁磁元件产生恒定磁场 → 椭球球心偏离原点
- **软铁干扰**：周围铁磁材料扭曲地磁场 → 球体被拉伸/旋转成椭球

本次标定通过**六位置静态采集**（加速度计）和**椭球拟合**（磁力计），利用地球重力（1g）和地磁场作为参考基准，求解误差模型参数。

## 2. 加速度计标定

### 2.1 标定方法

加速度计误差模型：

$$
\mathbf{a}_{cal} = M \cdot (\mathbf{a}_{raw} - \mathbf{b})
$$

其中 $\mathbf{a}_{raw}$ 为原始 ADC 值，$M$ 为 3×3 刻度因子矩阵，$\mathbf{b}$ 为 3×1 零偏向量。

**采集条件**：

- 传感器：MPU6050
- 采样率：约 50 Hz
- 每个姿态采集 5 秒
- 6 个姿态：+X, -X, +Y, -Y, +Z, -Z 分别朝上

**标定算法**：线性最小二乘法（正反面对求零偏和刻度）

### 2.2 标定结果

| 参数 | 数值 |
| :-- | :-- |
| **M 矩阵** | `[[1.00158, 0, 0], [0, 1.00409, 0], [0, 0, 1.00660]]` |
| **零偏 b** | `[-311.77, 103.26, 329.32]` |

**物理含义**：

- 三个轴的刻度因子均接近 1，误差 < 0.7%，传感器精度良好
- 零偏在几百 ADC 范围内（约 0.02~0.03g），属于 MPU6050 典型水平
- 非对角线元素为 0，说明轴间非正交误差可忽略


### 2.3 标定前后对比

| 指标 | 标定前 | 标定后 |
| :-- | :-- | :-- |
| 合加速度模长范围 | 约 15000~17300 ADC（约 0.92~1.06g） | 约 16200~16550 ADC（约 0.99~1.01g） |
| 模长标准差 | 约 450 ADC | 约 80 ADC |
| 与 1g 的偏差 | ±5% | ±0.5% |

**结论**：标定后合加速度模长显著收敛于理论值（16384 ADC），数据质量提升约 10 倍。

## 3. 磁力计椭球拟合标定

### 3.1 标定原理

磁力计误差模型：

$$
\mathbf{m}_{cal} = A^{1/2} \cdot (\mathbf{m}_{raw} - \mathbf{c})
$$

其中 $\mathbf{c}$ 为硬铁偏置（椭球球心），$A$ 为软铁变换矩阵。

**采集条件**：

- 传感器：HMC5883L
- 采样率：约 20 Hz
- 采集方式：手持传感器在空中缓慢画“8”字并不断翻转
- 有效点数：583 个

**标定算法**：直接最小二乘（DLS）椭球拟合 + SVD 分解

### 3.2 标定结果

| 参数 | 数值 |
| :-- | :-- |
| **硬铁偏置（球心）** | `[-19.48, -240.04, 79.77]` |
| **软铁矩阵** | `[[694.09, 63.27, 93.76], [63.27, 632.92, 105.19], [93.76, 105.19, 766.35]]` |

**物理含义**：

- 硬铁偏置表明存在较大的 Y 轴负方向恒定磁场干扰（约 -240）
- 软铁矩阵对角线元素反映了三个轴的灵敏度差异，非对角线元素反映了轴间耦合
- 标定后数据将恢复为以原点为中心的球体分布
