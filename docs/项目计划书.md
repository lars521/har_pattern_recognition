# 项目计划书 - 人体活动识别（HAR）

## 1. 任务目标
使用 ESP32 + MPU6050 + HMC5883L 采集腰部运动数据，构建一个能区分 6 类日常活动的模式识别系统，并通过被试独立评估验证泛化能力。

## 2. 类别集
Ω = { ω1:静坐, ω2:站立, ω3:步行, ω4:跑步, ω5:上楼, ω6:下楼 }

## 3. 样本 x 的定义
- 窗口长度：2.56 秒（128 点 @ 50 Hz）
- 重叠率：50%
- 每个窗口包含：ax, ay, az, gx, gy, gz, mx, my, mz（9 通道）

## 4. 特征空间
初步计划提取时域（均值、方差、偏度、峰度、过零率、SMA 等）和频域（主频、谱熵、频带能量）特征，维度约 30~50。

## 5. 决策函数 g(x)
采用贝叶斯分类器、线性判别、SVM、随机森林等，最终选择交叉验证下性能最好的模型。

## 6. 风险与代价
- 最小错误率贝叶斯（默认）
- 进阶：对“跌倒”类（如有）设更高漏检代价

## 7. 评估方法
- 留一被试（LOSO）交叉验证
- 指标：准确率、混淆矩阵、宏平均 F1
- 显著性检验：McNemar 检验
- 置信区间：自举法

## 8. 时间安排
严格按照实训任务书 D1~D10 执行。
